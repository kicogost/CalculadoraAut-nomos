# community-signal

Public forum threads (Reddit, etc.) from the recruiting posts — captured for the
discovery signal they carry *before/around* the 1:1 interviews. These are **public,
pseudonymous handles**, so they're committable (unlike the private interview
transcripts, which stay gitignored and pseudonymised).

Treat this as **directional context**, not interview data — comments are short and the
samples are channel-biased (e.g., expat subs skew toward "get a gestor"). The real
depth comes from the booked calls → see `../SYNTHESIS.md`.

One file per thread: `YYYY-MM-DD-<sub>-<topic>.md`.
